- ASP.NET Core Web API
- React + TypeScript
- PostgreSQL
- Railway
- Vercel

- TypeScript

- Backend → Railway

## 📌 Project Goal

WorkFlowPro was built as a production-style SaaS project management platform to demonstrate full-stack architecture, authentication flows, role-based access control, and modern deployment practices.
